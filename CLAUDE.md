# Luksuzne vikendice Fruška gora — projektne napomene

- Sve izmene koje korisnica traži odnose se SAMO na mobilnu verziju (≤880px, obično ≤600px preko media query-ja), osim ako izričito ne kaže drugačije. Desktop layout ostaviti netaknut.
- Korisnica obeležava elemente crtežom/komentarom deo po deo; primeniti izmenu tačno na obeleženi element.
- Mobilni pregled svih strana je u fajlu "Mobilni pregled.html" (telefonski okviri 390px).
